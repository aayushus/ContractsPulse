## 2026-06-11 - Interactive Element Accessibility\n**Learning:** Svelte's a11y linter strictly enforces keyboard accessibility. A `div` with an `onclick` handler must have `role="button"`, `tabindex="0"`, and an `onkeydown` handler for Enter and Space keys to be accessible to keyboard and screen reader users.\n**Action:** Always add proper ARIA roles, tabindex, and keyboard event handlers when creating custom interactive elements (like dropzones) that aren't native buttons.
